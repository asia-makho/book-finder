# Лабораторна робота №6: Book Finder (TypeScript)

Цей проєкт — веб-застосунок для пошуку книг через відкрите API OpenLibrary. Логіку переписано з JavaScript на TypeScript із використанням модульної структури.

## Запуск проєкту

```bash
# 1. Перейти в папку проєкту (замініть шлях на свій)
cd "шлях/до/lab5-bookfinder"

# 2. Встановити залежності (завантажити потрібні npm-пакети)
npm install

# 3. Зібрати проєкт (скомпілювати TypeScript-код у JavaScript)
npm run build

Відкриття у браузері
Відкрийте папку проєкту у VS Code.

Переконайтеся, що у вас встановлено розширення Live Server.

Відкрийте файл index.html.

Натисніть кнопку Go Live у правому нижньому куті екрана.
